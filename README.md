## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Unnati-singh-ai/task-manager.git
```

### Backend Setup

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
